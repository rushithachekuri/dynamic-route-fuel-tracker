Dynamic Route + Fuel Cost Tracker
📌 Problem Statement

Delivery workers often lose money due to inefficient routes, heavy traffic, and rising fuel prices. Most navigation apps suggest only the fastest route, not the most fuel-efficient route. Because of this, delivery workers end up spending more money on fuel every day.

Our solution helps workers choose routes that minimize fuel cost while maintaining efficiency, helping them save money and work smarter.

💡 Solution Overview

Dynamic Route + Fuel Cost Tracker is a smart system designed for delivery workers. The application analyzes multiple routes between two locations and calculates the fuel cost for each route. It allows users to compare the fastest route and the cheapest route, enabling better decision making.

The system also tracks fuel expenses and provides weekly savings insights, helping workers understand how much money they saved by choosing optimized routes.

👤 Persona-Based Scenario

Persona: Ramesh, a 26-year-old delivery worker in Vijayawada.

Challenges

Works 10–12 hours daily

Faces heavy traffic during deliveries

Petrol prices are increasing frequently

Needs a simple way to reduce fuel expenses

Workflow

1️⃣ Worker enters the destination in the app
2️⃣ The system fetches multiple routes using Google Maps API
3️⃣ Backend calculates the fuel cost for each route
4️⃣ The app shows Fastest Route vs Cheapest Route
5️⃣ Worker selects the best route
6️⃣ Weekly dashboard shows total savings

🧠 AI/ML Integration

Our system integrates Artificial Intelligence to improve route selection.

Route Optimization

Machine learning models analyze routes based on distance, time, and fuel consumption.

Fuel Cost Prediction

Uses historical fuel prices and real-time price data to estimate cost accurately.

Savings Forecast

Predicts weekly and monthly savings for delivery workers.

💰 Weekly Premium Model

Premium calculations are based on parametric triggers:

Distance traveled

Fuel price fluctuations

Traffic congestion

The AI system adjusts the premium dynamically so delivery workers are fairly compensated for increased operational costs.
